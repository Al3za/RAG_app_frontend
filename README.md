## Overview

**RAG_app_frontend** is a modern, Next.js-based web application designed to deliver a secure, scalable, and interactive user experience. It integrates seamless authentication, document ingestion, and AI-powered chat functionalities within a robust architecture optimized for deployment and performance.

**Why RAG_app_frontend?**

This project empowers developers to rapidly build and deploy dynamic, user-centric interfaces with ease. The core features include:

- 🧩 **🟣 Standalone Deployment:** Configured for optimized, self-contained builds suitable for serverless environments.
- 🌐 **🟢 Secure Authentication:** Integrates NextAuth.js with Google OAuth for seamless, secure user login.
- 📄 **🟠 PDF Upload & Processing:** Facilitates real-time document ingestion with status polling and backend integration.
- 💬 **🔵 AI Chat Interface:** Enables interactive, real-time conversations with AI models, supporting personalized user engagement.
- 🚦 **🟡 Monitoring & Health Checks:** Built-in endpoints for uptime monitoring and system health verification.
- 🔒 **🟤 Custom Session Management:** Extends session properties for secure token handling across the app.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** TypeScript
- **Package Manager:** Npm

### Installation

Build RAG_app_frontend from the source and install dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/Al3za/RAG_app_frontend
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd RAG_app_frontend
    ```

3. **Install the dependencies:**

**Using [npm](https://www.npmjs.com/):**

```sh
❯ npm install
```

### Usage

Run the project with:

**Using [npm](https://www.npmjs.com/):**

```sh
npm run start (or nmp run dev for development)
```
